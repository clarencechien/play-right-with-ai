## 使用方法

### 本地執行
1. 將所有檔案下載到同一目錄
2. 在瀏覽器中開啟 `index.html`
3. 開始使用應用程式

### 功能操作

#### 新增任務
1. 在輸入框中輸入任務名稱
2. 選擇優先級（選填）
3. 輸入描述（選填）
4. 設定到期日（選填）
5. 點擊「新增任務」按鈕

#### 編輯任務
- 點擊任務旁的編輯圖示 ✏️
- 直接修改任務標題
- 按 Enter 儲存或 Esc 取消

#### 刪除任務
- 點擊任務旁的刪除圖示 🗑️
- 確認刪除對話框

#### 篩選與搜尋
- 使用頂部按鈕篩選狀態
- 使用優先級按鈕篩選
- 在搜尋框輸入關鍵字

#### 批次操作
- 選擇多個任務
- 使用底部工具列進行批次操作

#### 資料管理
- 點擊「匯出資料」下載 JSON 檔案
- 點擊「匯入資料」上傳 JSON 檔案
