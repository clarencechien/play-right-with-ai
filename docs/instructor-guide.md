# Play Right with AI - 講師指南

## 概述

本指南為講師提供完整的教學框架，幫助您有效地傳授 AI 驅動開發工作流程。

## 教學理念

### 核心原則
1. **實作優先**：每個概念都透過實際操作學習
2. **漸進式學習**：從簡單到複雜，循序漸進
3. **即時反饋**：利用 AI 提供即時指導
4. **自主探索**：鼓勵學員實驗和創新

### 目標學員
- 具備基礎 JavaScript 知識
- 熟悉終端機操作
- 對 AI 工具有興趣但經驗有限
- 希望提升開發效率

## 常見學員問題與解答

### 環境設置問題

**Q1：Node.js 版本不相容怎麼辦？**
```bash
# 使用 nvm 管理版本
nvm install 18
nvm use 18
```

**Q2：MCP 連接失敗？**
- 檢查防火牆設置
- 確認瀏覽器版本
- 重啟 MCP 服務

### AI 使用問題

**Q3：AI 生成的代碼品質不佳？**
- 優化提示詞的具體性
- 提供更多上下文
- 使用範例引導
- 嘗試不同 AI 模型

**Q4：Token 限制問題？**
- 分割大型任務
- 使用摘要技巧
- 清理對話歷史
- 考慮付費方案

### 測試相關問題

**Q5：Playwright 測試不穩定？**
```javascript
// 加入等待策略
await page.waitForLoadState('networkidle');
await page.waitForSelector('#element', { timeout: 30000 });
```

**Q6：如何處理動態內容？**
- 使用適當的等待方法
- 實施重試機制
- 考慮資料注入

## 疑難排解指南

### 常見錯誤與解決方案

#### 1. 環境問題
```bash
# 權限問題
sudo npm install -g playwright

# 快取清理
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```

#### 2. AI API 問題
```javascript
// 錯誤處理範例
try {
  const response = await ai.generate(prompt);
} catch (error) {
  if (error.code === 'RATE_LIMIT') {
    await delay(60000); // 等待 1 分鐘
    return retry();
  }
}
```

#### 3. 測試失敗
```javascript
// 調試技巧
await page.screenshot({ path: 'debug.png' });
await page.pause(); // 手動調試
console.log(await page.content()); // 檢查 HTML
```
