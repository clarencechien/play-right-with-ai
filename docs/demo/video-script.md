## 腳本使用說明

### 語調指引
- 保持熱情但專業
- 用「我們」而非「你們」，建立連結
- 適時加入驚喜感（「看！」、「太棒了！」）
- 技術詞彙保持英文，解釋用中文

### 節奏控制
- Demo 時稍微放慢，確保觀眾跟上
- 概念說明保持輕快
- 重要觀念重複 2-3 次
- 適時停頓讓資訊沉澱

### 視覺提示
- 🎬 需要螢幕錄影的部分
- 📊 需要圖表或動畫的部分
- ⚡ 需要特效或轉場的部分
- 🎵 建議背景音樂變化的時機

### 後製建議
- 加入章節時間戳
- 程式碼片段使用語法高亮
- 重要概念使用字幕強調
- Demo 部分可適度加速（1.5x）
