## 投影片設計規範

### 色彩方案
```css
:root {
  --primary: #6366F1;     /* Indigo - 主色 */
  --secondary: #8B5CF6;   /* Purple - 輔色 */
  --success: #10B981;     /* Green - 成功 */
  --warning: #F59E0B;     /* Yellow - 警告 */
  --danger: #EF4444;      /* Red - 錯誤 */
  --dark: #1F2937;        /* Gray - 深色文字 */
  --light: #F9FAFB;       /* White - 淺色背景 */
}
```

### 字體建議
```css
.title {
  font-family: 'Noto Sans TC', sans-serif;
  font-weight: 700;
  font-size: 48px;
}

.subtitle {
  font-family: 'Noto Sans TC', sans-serif;
  font-weight: 500;
  font-size: 32px;
}

.body {
  font-family: 'Noto Sans TC', sans-serif;
  font-weight: 400;
  font-size: 24px;
}

  font-family: 'Fira Code', monospace;
  font-size: 20px;
}
```

### 動畫效果
```yaml
進入效果:
  - 淡入 (Fade In)
  - 滑入 (Slide In)
  - 縮放 (Scale)
  - 彈跳 (Bounce)

強調效果:
  - 脈動 (Pulse)
  - 搖晃 (Shake)
  - 高亮 (Highlight)
  - 閃爍 (Blink)

轉場效果:
  - 交叉淡化 (Cross Fade)
  - 推移 (Push)
  - 覆蓋 (Cover)
  - 翻頁 (Page Turn)
```

### 版面配置原則
1. **視覺層次**：標題 > 副標題 > 內文
2. **留白空間**：至少 20% 空白區域
3. **對齊方式**：置中對齊為主
4. **圖文比例**：40% 圖像、60% 文字
5. **色彩使用**：每頁不超過 3 種主色

### 投影片製作工具推薦
- **Keynote** (macOS)：精美模板、流暢動畫
- **PowerPoint** (跨平台)：功能完整、普及度高
- **Google Slides** (網頁)：協作方便、免費使用
- **Reveal.js** (網頁)：程式碼友善、可版本控制
- **Figma** (設計)：精細設計、可匯出各種格式
