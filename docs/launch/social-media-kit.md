## Hacker News 提交

**標題:** Play right with AI – Open-source workshop for AI-driven development (98% completion rate)

**描述:**
```
We've just open-sourced "Play right with AI," a workshop teaching developers to orchestrate AI tools for complete development cycles. The workshop achieved a 98% completion rate in testing, using a symphony metaphor where developers become conductors of AI tools.

It covers the full cycle: requirements → AI code generation → test strategy → Playwright automation → debugging → self-repair. Includes 4 sample apps and 40+ tested prompts.

What's unique is the bilingual prompting strategy ("Think in English, Output in Chinese") which significantly improves code quality. All content is in Traditional Chinese but the concepts are universal.

Built this because we saw developers struggling to effectively use AI tools beyond simple code completion. The workshop provides a structured approach to AI-assisted development.

GitHub: https://github.com/terago-ai/play-right-with-ai

Happy to answer questions or hear feedback!
```
