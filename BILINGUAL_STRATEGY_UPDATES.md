## 🎯 核心理念

### 為什麼採用雙語策略？

1. **技術精確度** - AI 在英文環境下推理能力更強
2. **邏輯清晰度** - 英文結構有助於組織技術思路
3. **在地化需求** - 中文輸出確保本地開發者理解
4. **最佳效果** - 結合兩種語言優勢

### 基本模式

```
[English Thinking]
- Analyze requirements
- Design architecture  
- Consider edge cases
- Plan implementation

[Chinese Output]
請用繁體中文輸出完整解決方案
```
