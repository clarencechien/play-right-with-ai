## 專案挑戰總覽

### 🎯 核心挑戰：線上學習平台

你將獨立使用 AI 工具，從零開始構建一個完整的線上學習平台，包含：

**功能需求**：
- 用戶註冊和登入系統
- 課程瀏覽和搜尋
- 影片播放和進度追蹤
- 測驗和作業提交
- 討論區和即時聊天
- 個人學習儀表板

**技術要求**：
- 前端：現代 JavaScript 框架
- 後端：RESTful API 或 GraphQL
- 資料庫：關聯式或 NoSQL
- 測試：>80% 覆蓋率
- 部署：容器化部署方案
