# Project Brief: Play right with AI Workshop

## Project Overview
An open-source online workshop teaching AI-driven self-cycling development and testing workflows using Playwright and AI orchestration.

## Core Purpose
Transform developers into "AI Conductors" who can orchestrate complete development cycles using AI tools - from application generation to self-repair.

## Key Requirements
1. **Workshop Structure**: 8 progressive chapters from setup to capstone project
2. **Bilingual Strategy**: "Think in English, Output in Chinese" prompting approach
3. **Self-Cycling Workflow**: AI generates → tests → debugs → repairs automatically
4. **GitHub-Based**: All materials in public repository for community learning
5. **Language**: Traditional Chinese (繁體中文) for all learner-facing content

## Target Audience
- Forward-thinking developers embracing "Vibe Coding"
- Test engineers wanting to leverage AI
- DevOps engineers integrating AI into workflows
- Technical leaders interested in next-gen development paradigms

## Success Metrics
- 500+ GitHub stars within 6 months
- 100+ forks within 6 months
- 50+ community discussions
- 100+ learners completing capstone project

## Technical Stack
- **Testing**: Playwright
- **AI Tools**: Claude, Gemini, GPT-5
- **Integration**: Playwright MCP (Model Context Protocol)
- **Languages**: JavaScript/TypeScript
- **Environment**: VS Code, Node.js

## Deliverables
1. `/prompts/` - Golden prompts collection
2. `/workshop/` - Chapter exercises and materials
3. `/sample-app-source/` - Example applications
4. `.claude/agents/` - Specialized AI agents
5. Documentation in Traditional Chinese

## Constraints
- Assumes basic JavaScript/TypeScript knowledge
- Requires AI service API keys
- No deep programming language teaching
- No extensive manual coding required

## Project Timeline
- Phase 1: Foundation setup (Complete)
- Phase 2: Workshop content creation
- Phase 3: Community launch
- Phase 4: Iteration based on feedback
