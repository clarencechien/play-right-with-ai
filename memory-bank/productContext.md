# Product Context: Play right with AI Workshop

## Why This Project Exists

### The Problem
Developers treat AI as passive "code snippet generators" rather than active development partners. They lack systematic methodology to chain multiple AI tools into a closed-loop workflow that can autonomously complete development cycles.

### The Solution
A comprehensive workshop teaching developers to become "AI Conductors" - orchestrating AI tools to create self-cycling workflows that generate, test, debug, and repair applications automatically.

### The Vision
Enable the next efficiency revolution in software development by teaching developers how to leverage AI's full potential through systematic orchestration rather than ad-hoc usage.

## How It Should Work

### Learning Journey
1. **Mindset Shift** (Ch 1): From coding to conducting
2. **Generation** (Ch 2): AI creates applications from requirements
3. **Strategy** (Ch 3): AI plans comprehensive testing
4. **Automation** (Ch 4): AI writes Playwright tests
5. **Analysis** (Ch 5): AI diagnoses failures
6. **Repair** (Ch 6): AI fixes issues automatically
7. **Optimization** (Ch 7): Advanced techniques and patterns
8. **Mastery** (Ch 8): Independent orchestration

### Core Innovation: Bilingual Prompting
- **English Thinking**: Leverages AI's superior English reasoning
- **Chinese Output**: Ensures local developer accessibility
- **Hybrid Advantage**: Combines precision with usability
